# Projeto feito para aprender sobre MERN stack.
#### Neste projeto você pode criar uma conta, logar na sua conta e pesquisar restaurantes pelo nome, zipcode e pela cozinha.
#### Também pode criar, editar e excluir reviews do restaurante.
